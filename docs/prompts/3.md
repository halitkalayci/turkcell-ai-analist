## Planladığım Değişiklikler

products-v1.yml dosyasına aşağıdaki eklemeleri yapacağım:

### 1. **POST /products Endpoint'i Oluşturma**
   - **Path**: `/products`
   - **Method**: POST
   - **Açıklama**: Yeni ürün ekleme endpoint'i

### 2. **Request Body (CreateProductRequest) Şeması**
   ```yaml
   - name: 
     - type: string
     - minLength: 2
     - maxLength: 150
     - pattern: "^(?!\\s*$)\\S.*\\S$|^\\S$" (sadece boşluk kontrolü + trim kontrolü)
     
   - price:
     - type: number
     - format: double
     - minimum: 0.0
     
   - currency:
     - type: string
     - enum: ["₺", "$"]
   ```

### 3. **Response Şemaları**
   
   **ProductResponse (başarılı yanıt için)**:
   - id: UUID formatında string
   - name: string
   - price: number
   - currency: string
   
   **HTTP Status Kodları**:
   - **201 Created**: Ürün başarıyla oluşturuldu
   - **400 Bad Request**: Validasyon hataları (isim kuralları, fiyat, currency)
   - **409 Conflict**: Aynı isimde ürün zaten mevcut

### 4. **Error Response Şeması**
   - timestamp: string (ISO 8601)
   - status: integer
   - error: string
   - message: string
   - path: string

### 5. **Validation Kuralları (description'larda belirtilecek)**
   - İsim 2-150 karakter arası olmalı
   - İsim sadece boşluklardan oluşamaz
   - İsim önünde ve sonunda boşluk olamaz (trim)
   - Aynı isimde ürün eklenemez
   - Fiyat 0 veya daha büyük olmalı
   - Currency sadece ₺ veya $ olabilir
