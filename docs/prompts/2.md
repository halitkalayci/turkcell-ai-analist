# Mikroservis E-Ticaret Projesi Taslağı

## Kullanılacak Teknolojiler

### Backend
- **Java 17+**
- **Spring Boot 3.X**
- **PostgreSQL** (veritabanı)
- Spring Cloud (mikroservis altyapısı)
- Spring Security + JWT (kimlik doğrulama)
- Netflix Eureka (servis keşfi)
- API Gateway (Spring Cloud Gateway)
- Docker & Kubernetes (konteynerizasyon)
- RabbitMQ/Kafka (mesajlaşma)
- Redis (önbellekleme)

### Frontend
- **Node.js**
- **React**
- React Router (sayfa yönlendirme)
- Axios (HTTP istekleri)
- Redux/Context API (state yönetimi)

---

## Mikroservisler ve Özellikleri

### 1. **API Gateway Service**
- Tüm istekleri yönlendirme
- Rate limiting
- Kimlik doğrulama/yetkilendirme kontrolü
- Load balancing

### 2. **User Service (Kullanıcı Servisi)**
- Kullanıcı kayıt/giriş
- Profil yönetimi
- Şifre sıfırlama
- Rol ve yetki yönetimi (Admin, Müşteri, Satıcı)
- JWT token üretimi

### 3. **Product Service (Ürün Servisi)**
- Ürün CRUD işlemleri
- Kategori yönetimi
- Ürün arama ve filtreleme
- Stok takibi
- Ürün görselleri yönetimi
- Fiyat geçmişi

### 4. **Order Service (Sipariş Servisi)**
- Sipariş oluşturma
- Sipariş durumu takibi
- Sipariş geçmişi
- Sipariş iptali/iadesi
- Fatura oluşturma

### 5. **Payment Service (Ödeme Servisi)**
- Ödeme işlemleri
- Çoklu ödeme yöntemi desteği (Kredi kartı, havale, vb.)
- Ödeme durumu takibi
- Geri ödeme işlemleri
- Ödeme gateway entegrasyonu

### 6. **Inventory Service (Envanter Servisi)**
- Stok yönetimi
- Stok girişi/çıkışı
- Düşük stok uyarıları
- Depo yönetimi

### 7. **Cart Service (Sepet Servisi)**
- Sepete ürün ekleme/çıkarma
- Sepet güncelleme
- Sepet tutma (oturum kapatılsa bile)
- Sepet özetleme

### 8. **Notification Service (Bildirim Servisi)**
- E-posta bildirimleri
- SMS bildirimleri
- Push bildirimleri
- Sipariş durumu bildirimleri

### 9. **Review Service (Yorum/Değerlendirme Servisi)**
- Ürün yorumları
- Ürün puanlama
- Yorum moderasyonu
- Yanıt verme

### 10. **Search Service (Arama Servisi)**
- Elasticsearch entegrasyonu
- Gelişmiş arama
- Otomatik tamamlama
- Arama önerileri

### 11. **Service Discovery (Eureka Server)**
- Mikroservislerin birbirini bulması
- Load balancing
- Health check

### 12. **Config Server**
- Merkezi yapılandırma yönetimi
- Ortam bazlı konfigürasyonlar

## Ek Özellikler

### Güvenlik
- JWT tabanlı kimlik doğrulama
- OAuth2 entegrasyonu
- HTTPS zorunluluğu
- SQL injection koruması
- XSS koruması

### Performans
- Redis ile önbellekleme
- Database indexleme
- Pagination
- Lazy loading

### Monitoring & Logging
- Spring Boot Actuator
- ELK Stack (Elasticsearch, Logstash, Kibana)
- Prometheus & Grafana
- Distributed tracing (Zipkin/Jaeger)

### DevOps
- Docker containerization
- Kubernetes orchestration
- CI/CD pipeline (GitHub Actions)
- Automated testing

### Database Strategy
- Her mikroservis kendi PostgreSQL veritabanına sahip
- Database per service pattern
